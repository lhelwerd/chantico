## Upcoming milestones

We document some of our ideas for upcoming milestones. This content is still up for debate.

### Milestone 2

#### Context of goals

- We want to further configure chantico for testing in the HESI lab.
- We want to deploy an initial version of the chantico package to Staging lab.
- We want to move everything to GitHub (pipeline, issues and milestone planning, 
  documentation website deployment, container registry, etc.)

#### Features under consideration

- We want to include VM monitoring. Specific hypervisors to support are not yet 
  decided, but this may be OpenNebula and/or Proxmox.
- We want to provide more integration capabilities, such as an API reference for 
  a service orchestrator or other cloud platform tooling to 
  create/read/update/delete the resource information.
